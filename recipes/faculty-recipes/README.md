# Faculty projects — Day 1 recipe seeds

Project scaffolds grown from the Day-1 "project as recipe" cards and from two
faculty who emailed ideas instead of filling out a card. Each folder is a
starter project with the same shape: **`context/`** (inputs + distilled source
notes + a starter `CLAUDE.md`), **`prompts/`** (reusable prompt templates), and
**`outputs/`** (a representative sample output). `README.md` in each folder holds
the recipe spec.

People who emailed are reduced to **initials** (JG, JB); recipe cards were
anonymous and are named by project. Raw captures stay archived in
`../day-1-recipe-images/` and `../day-1-transcripts/`.
