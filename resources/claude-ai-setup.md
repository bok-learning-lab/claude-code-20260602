# Claude.ai Setup Guide

For the workshop, you'll use **Claude.ai** — Anthropic's web interface. No installation required.

---

## Getting started

1. Go to [claude.ai](https://claude.ai)
2. Sign in with your Google account, or create a free Anthropic account with your email
3. You'll land on the chat interface — you're ready to go

---

| Plan | Cost | What it gives you |

---

## Key features for today's workshop

### Uploading files
Click the **paperclip icon** (or drag and drop) to upload:
- Images (JPG, PNG, WEBP, GIF)
- Documents (PDF, Word, text files)
- Code files

Claude can read and analyze everything you upload.

### Switching models
Use the model selector at the top of the chat window to switch between:
- **Claude Sonnet** — fast, great for most tasks
- **Claude Opus** — most capable, best for complex analysis

### Projects
Under **Projects** in the sidebar, you can create a persistent workspace where Claude remembers context across conversations. Useful for ongoing work with the same codebase, document set, or research project.

---

## Tips for today

- **You can attach multiple files at once** — drag and drop them all in before sending your first message
- **Claude remembers the full conversation** — you don't need to re-explain context on every message
- **Be specific in your requests** — describe the format and detail level you want in the output
- **Iterate** — if the first result isn't right, follow up and redirect
